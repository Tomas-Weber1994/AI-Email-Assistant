Hiring Task
AI Email & Calendar Assistant
Task
Build an AI agent that monitors a Gmail inbox, classifies incoming emails, and takes appropriate autonomous actions — including calendar coordination and human approval where needed.
You will set up a dedicated Gmail account and Google Calendar for this task and share the credentials with us before the evaluation. We will test by running the agent and sending 20 emails to the inbox; your agent must handle them without any prior knowledge of their content.
📅  The calendar will be pre-populated with existing events. The agent must plan around them.

Some emails should receive an automatic reply — for example, a polite acknowledgement to a marketing offer or a brief thank-you where appropriate. The agent decides when this is warranted.
Calendar events must respect existing availability and be planned reasonably. Any calendar action or reply sent on behalf of the manager requires email confirmation first: the agent sends a summary to the manager, who replies APPROVE or REJECT.
Non-Functional Requirements
Audit trail: every processed email must produce a structured log entry (label, action, outcome).
Error handling: the agent must fail gracefully and log any failures.
Human-in-the-loop: manager approval via email reply before any send or calendar action.
Stack
Any Python framework/custom code and any LLM API. Gmail and Google Calendar integrations are required.
💡  If the agent is built to use Azure OpenAI, we can substitute our own AIS credentials during testing.
Deliverables
Gmail and Google Calendar credentials (set up by you).
A dockerized agent connected to the inbox, ready for live evaluation.
